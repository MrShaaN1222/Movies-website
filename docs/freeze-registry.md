# Freeze Registry

Tracks approved modules that are frozen (locked) and cannot be changed except approved hotfix policy.

## Lock Entry Template

LOCKED: `<module-id>` | `<phase>` | `<approved-by>` | `<date>` | `<tag>`

- Scope: `<files/services owned by module>`
- Test report: `<path>`
- API/schema snapshot: `<path>`
- Rollback plan: `<path>`
